# The Rehearsal Calendar

A civic instrument that measures time by communal return rather than duration.

Most calendars tell us where we are in time. This one asks whether something has continued.

It tracks drift — the growing distance between a practice and the last act that renewed it. When communities return to what they carry, the calendar moves. Time advances not because days have passed but because people have acted.
